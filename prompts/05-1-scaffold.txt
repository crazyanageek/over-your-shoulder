=== GLOBAL LAYOUT CONSTRAINTS ===

**Format**: A4 portrait, 210mm × 297mm.
**Margins**: inner padding of 18mm top, 18mm bottom, 16mm left, 16mm right (give the page a real margin that matches professional documents).
**Dimensions**: use `@page { size: A4 portrait; margin: 0; }` in CSS, then set each `.page` to `width: 210mm; height: 297mm` with padding inside.
**Page break**: `page-break-after: always` on each `.page` except the last.
**Background**: `--bg #FAFAF7` (same as badge).
**Text color**: `--ink #0E0E0D`.
**Fonts**: bundle Fraunces (titles), Inter (body), JetBrains Mono (data/code) — REUSE THE FONT FILES already bundled in `/report/fonts/` from phase 04.5.2. Do NOT re-download them, do NOT link to Google Fonts.

**CSS variables** (at root of full.css, same palette as the badge):
:root {
--bg: #FAFAF7;
--ink: #0E0E0D;
--ink-muted: #666;
--ink-soft: #999;
--rule: #E8E5DD;
--alarm: #B91C1C;

--make-volume: #2B2B2A;
--make-cat: #BFA57A;
--make-geo: #B91C1C;

}

**Typography scales**:
- H1 (page title): Fraunces italic, 32pt, line-height 1.1, ink color
- H2 (section title): Fraunces italic, 20pt, line-height 1.2, ink color
- H3 (sub-section): Inter 600 uppercase, 10pt, letter-spacing 0.15em, ink-muted color
- Body: Inter 400, 11pt, line-height 1.55, ink color
- Data/mono: JetBrains Mono 400, 10pt, line-height 1.5, ink color
- Caption/hint: Inter 400 italic, 10pt, ink-muted color
- Footer text: Inter 400, 9pt, ink-soft color

**Horizontal rules**: 1px solid `--rule`, used as section separators, never for decoration.
