=== Create lib/score.js ===

Export a pure function computeScore(counters, daysActive) that returns:

{
  score: Number (0-100),
  breakdown: {
    volume: Number (0-30),
    categoryDiversity: Number (0-30),
    geoExposure: Number (0-25),
    continuity: Number (0-15)
  },
  meta: {
    totalRequests: Number,
    categoriesDetected: Array<string>,
    countries: Array<string>,
    daysActive: Number,
    usEquivalentShare: Number (0-1)
  }
}

Formulas:

F1 Volume (30 pts max):
Math.min(30, 5 * Math.log10(totalRequests + 1))
Caps out around 1000+ req/day.

F2 Category diversity (30 pts max):
5 points per distinct sensitive category detected in bySourceCategory, among: email, code, documents, communications, crm, finance, professional.
Do NOT count "other".

F3 Geographic exposure (25 pts max):
Weight each country by sensitivity:
- US = 1.0
- CN = 1.0
- UK = 0.6
- CA = 0.5
- EU (FR, DE, IT, ES, NL, BE, SE, IE, PL, etc.) = 0.4
- CH = 0.1
- unknown = 0.7 (default-high when we can't determine)
usEquivalentShare = sum(weight * count) / totalRequests
geoExposure = 25 * usEquivalentShare

F4 Continuity (15 pts max):
15 * Math.min(1, daysActive / 7)
where daysActive = number of distinct oys_events_YYYYMMDD keys with at least 5 events.
