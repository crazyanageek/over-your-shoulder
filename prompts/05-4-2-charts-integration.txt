=== DATA PREPARATION ===

In full.js, after buildTokens() is called and the context is resolved, build 4 data objects for the charts:

1. vendorsData:
   From counters.byVendor, build array of { name, count, share } sorted descending by count. Apply the same OpenAI→ChatGPT remap used in buildVendorList.

2. hourlyData:
   Array of 24 values (0-23), count of events per hour of day. Iterate today's oys_events_YYYYMMDD entries (or all events if today-specific not computable), bucket by local hour.

3. scoreHistoryData:
   Direct pass-through of oys_score_history (already sorted asc by date).

4. mapData:
   origin = COUNTRY_COORDS[userCountry].coords (same as badge map)
   destinations = array of { city, country, coords, count } derived from counters.byHost. For each host in byHost, look up HOST_GEO for city/country/coords, aggregate by unique city (sum counts if multiple hosts map to same city — e.g., openai.com + chatgpt.com both → San Francisco).
