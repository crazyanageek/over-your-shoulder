=== DESIGN PRINCIPLES (apply to all 4 charts) ===

Option C aesthetic — mostly monochrome, alarm red (#B91C1C) used as ACCENT only to highlight the most critical value in each chart.

Principle 1: Editorial sobriety
- Black/grey bars and lines by default
- Thin strokes (0.5-1.5px)
- No drop shadows, no gradients, no animations
- Fraunces italic for chart titles, Inter regular for labels

Principle 2: Minimum annotations
- Axis labels: only if essential for comprehension
- Value labels on bars: only on the top-ranked value, or on the "alarm" value
- No legend unless multiple series need disambiguation
- Let the chart shape communicate; the micro-list below provides precise numbers

Principle 3: Alarm red as accent
- Use --alarm (#B91C1C) for the SINGLE most critical value in each chart:
  - Vendors bar: the top vendor
  - Hourly histogram: the peak hour bar
  - Score evolution: today's point (marker at the end of the line)
  - Map: the origin (user) marker stays --ink, but the arcs remain grey (no red arcs — too loud)
- All other bars/lines/points: --ink-muted or --ink-soft
