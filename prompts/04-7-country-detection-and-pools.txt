=== PART 2 — Country detection logic ===

Create a new file /lib/country.js (or add to existing lib/ if you prefer) exporting:

  async function detectUserCountry() { ... }

Logic in order of priority:

1. Check chrome.storage.local.oys_user_country:
   - If missing OR equals "auto" → continue to auto-detection
   - If equals a country code ("CN", "FR", "DE", "CH", "GB", "US") → return that code
   - If equals "OTHER" → return null (triggers neutral pool)

2. Auto-detect using timezone:
   Get Intl.DateTimeFormat().resolvedOptions().timeZone (e.g. "Europe/Zurich")
   Map common timezones to country codes:
   
   const TZ_MAP = {
     "Europe/Zurich": "CH",
     "Europe/Paris": "FR",
     "Europe/Berlin": "DE",
     "Europe/Frankfurt": "DE",
     "Europe/Munich": "DE",
     "Europe/London": "GB",
     "America/New_York": "US",
     "America/Chicago": "US",
     "America/Denver": "US",
     "America/Los_Angeles": "US",
     "America/Phoenix": "US",
     "America/Anchorage": "US",
     "Pacific/Honolulu": "US",
     "Asia/Shanghai": "CN",
     "Asia/Beijing": "CN",
     "Asia/Hong_Kong": "CN",
     "Asia/Chongqing": "CN",
     "Asia/Urumqi": "CN"
   };
   
   If timezone matches, return the code.

3. Fallback to language detection:
   Use navigator.language (e.g. "fr-CH", "en-US"). Extract the country suffix after the dash.
   Only trust this if it matches our 6 supported countries; otherwise proceed to step 4.

4. If neither worked, return null.

The function is called by report.js at render time.

SELECTION LOGIC in report.js:

Replace the existing usShare-based selection with:

  async function selectMapHeadline(punchlines, counters, totalRequests) {
    const userCountry = await detectUserCountry();
    const byCountry = counters.byCountry || {};
    const countryEntries = Object.entries(byCountry);
    if (countryEntries.length === 0) return pickRandom(punchlines.mapHeadline.en.neutral);
    
    // Find dominant country (>= 70% of traffic)
    const sortedCountries = countryEntries.sort((a, b) => b[1] - a[1]);
    const [topCountry, topCount] = sortedCountries[0];
    const topShare = topCount / totalRequests;
    
    if (topShare < 0.70) {
      // No dominant country
      return pickRandom(punchlines.mapHeadline.en.neutral);
    }
    
    // There IS a dominant country
    if (userCountry === topCountry) {
      // User is IN the dominant country — domestic pool
      const domesticPool = punchlines.mapHeadline.en.domestic[topCountry];
      if (domesticPool) return pickRandom(domesticPool);
      // Fallback: no domestic pool for this country
      return pickRandom(punchlines.mapHeadline.en.neutral);
    } else {
      // User is OUTSIDE the dominant country — dominantByCountry pool
      const dominantPool = punchlines.mapHeadline.en.dominantByCountry[topCountry];
      if (dominantPool) return pickRandom(dominantPool);
      // Fallback: no dominant pool for this country
      return pickRandom(punchlines.mapHeadline.en.neutral);
    }
  }

Token substitution: {N} is the count for the dominant country (NOT total). {C} is the number of distinct countries (for neutral pool only, where it appears).
